# Intia

Intia est une application web composée d’un Backend (Node.js / Express / MongoDB) et d’un frontend (React.js).  
NB: Le backend doit être lancé avant le frontend pour que l’application fonctionne correctement.

-------------------------------------------------------------------------------------------------------------

# Prérequis
Assurez-vous d’avoir installé sur votre PC :

- Node.js : version 24.11.1  
- Vérification : node --v (sur le terminal de votre PC)

-------------------------------------------------------------------------------------------------------------

# Structure du Projet
Intia/
├── Intia-Backend
├── Intia-Frontend
└── DOC-INTIA

-------------------------------------------------------------------------------------------------------------

# Lancement du Backend
cd Intia/Intia-Backend
npm install
npm run dev

-------------------------------------------------------------------------------------------------------------

# Lancement du Frontend
cd Intia/Intia-Frontend
npm install
npm run dev
